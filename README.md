# Backend_LH

Project was built and tested with Python 3.12.3.

# Dependencies:

**Fastapi**

> pip install fastapi

**Uvicorn**

> pip install uvicorn

**Pydantic**

> pip install pydantic

# Instructions

The service should start running at : http://127.0.0.1:8000/

To start the API, run:

> uvicorn main:app
